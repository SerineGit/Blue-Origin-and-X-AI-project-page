Comprehensive Testing Framework for Blue Origin and X.AI Applications
This project delivers an end-to-end quality assurance solution for Blue Origin's website and X.AI Grock API. Our process covers UI automation and API testing, ensuring robust validation of functionality, performance, and reliability.

Website Testing: Manual & automated UI testing (Selenium), detailed test cases, posetive, negative & unittesting.
Blue Origin API Testing: Postman collections, REST API validation for accuracy, performance & error handling.
X.AI Grock API Testing: End-to-end API workflows, AI response verification, stability testing.
